# dashboard-analise-salarios-area-dados-imersao-python-alura

## Análise de Dados com Python – Projeto Imersão Dados (Alura)

Este projeto foi desenvolvido durante a Imersão Dados Python da Alura, com foco em todas as etapas essenciais de um pipeline de análise de dados. O objetivo foi explorar um conjunto de dados do início ao fim, aplicando técnicas fundamentais para transformar dados brutos em insights visuais e interativos.

O trabalho iniciou-se com o carregamento dos dados, seguido pelo processamento, tratamento e limpeza, garantindo a consistência e a qualidade das informações utilizadas. Foram aplicadas etapas como verificação de tipos, remoção de valores ausentes, padronização e organização das variáveis para posterior análise.

A partir dos dados preparados, foram calculadas métricas estatísticas importantes para compreensão do comportamento do conjunto, como médias, distribuições, correlações e segmentações relevantes. Essas informações foram representadas por gráficos exploratórios, utilizando bibliotecas como pandas, matplotlib e seaborn, oferecendo uma visão clara das tendências e padrões encontrados.

Para tornar os resultados acessíveis e interativos, foi desenvolvido um dashboard em ambiente virtual utilizando Streamlit, que permite navegação dinâmica entre visualizações e métricas. A construção foi realizada tanto no Google Colab, para experimentação e desenvolvimento inicial, quanto no VS Code, para estruturação final da aplicação. Durante o processo, realizou-se a instalação e configuração das bibliotecas necessárias dentro de um ambiente virtual Python, garantindo organização do projeto e reprodutibilidade.

Este projeto demonstra domínio sobre todas as fases da análise de dados: desde o pré-processamento até a apresentação final dos insights em um dashboard funcional.

## Construindo um Dashboard com Streamlit

Aprender a usar a biblioteca Streamlit para a criação de um dashboard interativo simples, que permite visualizar dados filtrados e gerar gráficos de forma prática requer:

Criar o ambiente virtual: python3 -m venv .venv

Ativar o ambiente virtual em Windows: .venv\Scripts\Activate

Ativar o ambiente virtual em MAC/LINUX: source .venv/bin/activate

Criar um arquivo chamado requirements.txt e adicionar os pacotes necessários: pandas==2.2.3, streamlit==1.44.1, plotly==5.24.1

Instalar as bibliotecas necessárias: pip install -r requirements.txt

Criar a Interface do Dashboard com Streamlit

Realizar o deploy do Dashboard no Streamlit Cloud: https://streamlit.io/cloud
